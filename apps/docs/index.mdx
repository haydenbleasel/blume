# Blume

Blume is an open-source documentation framework shipped as the `blume` package.
It scans Markdown and MDX content, generates a hidden Astro project under
`.blume/`, and builds a documentation site with navigation, search, raw
Markdown routes, LLM files, optional OpenAPI or AsyncAPI references, optional
Ask AI, and optional MCP discovery.

## Core concepts

| Concept           | What Blume uses it for                                                          |
| ----------------- | ------------------------------------------------------------------------------- |
| Content root      | The Markdown and MDX files scanned into the content graph.                      |
| `meta.ts`         | Folder-level navigation metadata authored with `defineMeta`.                    |
| `.blume/`         | Generated Astro runtime, regenerated by Blume commands.                         |
| `blume.config.ts` | Site, content, navigation, search, AI, theme, and deployment configuration.     |
| Package exports   | Type-safe helpers such as `defineConfig`, `defineMeta`, and `defineComponents`. |

## What Blume generates

`blume build` creates the Astro output, `llms.txt`, `llms-full.txt`, search
artifacts, sitemap and robots files when configured, and rendered references for
enabled OpenAPI or AsyncAPI specs.
