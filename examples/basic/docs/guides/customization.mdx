# Customization

Blume scales from zero config to full ownership.

## Theme tokens

Add a `theme.css` to override CSS variables:

```css
:root {
  --blume-accent: oklch(0.68 0.14 180);
}
```

## Components

Override built-in components with a `components.ts` file using
`defineComponents`.
