# Introduction

Welcome to **Blume** — a markdown-first documentation framework powered by
Astro and Vite. Drop in Markdown or MDX, run `blume dev`, and get a polished
docs site with navigation, theming, and search.

## Why Blume

- Zero-boilerplate authoring: just a folder of docs.
- Astro/Vite performance with static-first output.
- Full open-source ownership and source-level customization.
